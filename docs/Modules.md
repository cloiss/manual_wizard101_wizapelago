# Modules
The modules system allows sections of Wizard City can be selectively disabled or reduced for a shorter game. Each template yaml will have different defaults for these modules. You can also go into the yamls yourself and change them to your liking.

## Settings

### Golem Court
`none` - No Golem Court goals.

`half` - Excludes all Golem Tower goals.

`full` - All Golem Court goals.

### Pet Pavilion
`none` - No Pet Pavilion goals.

`full` - All Pet Pavilion goals.

There is no `half` option for this module.

### Cyclops Lane
`none` - No Cyclops Lane goals.

`half` - Excludes all goals past the main quest "The Cave Escape".

`full` - All Cyclops Lane goals.

### Triton Avenue
`none` - No Triton Avenue goals.
