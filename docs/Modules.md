# Modules
The modules system allows sections of Wizard City to be selectively disabled or reduced for a shorter game. Three template yamls for short, medium, and long games using the Modules feature have been provided. You can also go into the yamls yourself and change them to your liking.

## Settings

### Golem Court
`none` - No Golem Court goals.

`half` - Excludes all Golem Tower goals.

`full` - All Golem Court goals.

### Pet Pavilion
This module is currently disabled due to a bug in the game preventing access to any of the quests in Pet Pavilion.

### Cyclops Lane
`none` - No Cyclops Lane goals.

`half` - Excludes all goals past the main quest "The Cave Escape" (Cyclops/Fodder/Bubbles). (Side quests unlocked by this quest, such as the Eyus Maximus quest, will still be included)

`full` - All Cyclops Lane goals.

### Triton Avenue
`none` - No Triton Avenue goals.

`half` - Excludes all goals past the main quest "The Shadow of Death Magic" (Fodders).

`full` - All Triton Avenue goals. (This will also include the two Duncan quests in Haunted Cave)

### Firecat Alley
`none` - No Firecat Alley goals.

`half` - Excludes all goals past the main quest "Which Hunt?" (Pathfinders/Magma Men). (Side quests unlocked by this quest, such as the Wormguts questline, will still be included)

`full` - All Firecat Alley goals.

### Post-Streets
`none` - No post-streets goals.

`full` - All post-streets goals. This refers to any quests that open up after completing all 3 streets, such as "Colossal Trouble".

There is no `half` option for this module.

### The Lost Pages
`none` - No Lost Pages goals.

`full` - Adds the Wizard City Lost Pages questline, for use when that event is active.
